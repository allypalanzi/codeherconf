# Living Style Guides

Demo/files from the Living, Breathing Style Guides session at Code(her) Conference 2014.

## Demo

### Running Middleman

`git clone`

`cd` to the repo

If you don't have bundler, `gem install bundler`

`bundle install`

`bundle exec middleman`

### Running Grunt

in another terminal tab, `cd` to the demo directory

run `grunt`
